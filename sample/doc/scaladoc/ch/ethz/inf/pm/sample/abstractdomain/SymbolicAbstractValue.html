SymbolicAbstractValue

class SymbolicAbstractValue [S <: State[S]] extends FunctionalDomain[Expression, S, SymbolicAbstractValue[S]]

The representation of an expression inside an abstract state of the analysis. Since we could have to consider several different expressions (e.g., because of nondeterministic statements) that are produced with different abstract states. We abstract it with a function that relates each expression to the abstract state in which the expression was generated.
